## 🎯 核心亮点

### 产品定位
> **专注于 PPE 垂直领域，打造小而美、小而精的合规信息服务平台**

- 覆盖全球 6+ 主要经济体
- 提供检索、文件生成、合规评估等一站式服务
- Free + Pro + Enterprise 三层订阅模式

### 技术架构
> **现代化、云原生、可扩展的技术栈**

- 前端：Next.js 16 + React 19 + TypeScript
- 后端：NestJS + PostgreSQL + Redis
- 部署：Docker + Vercel + AWS
- AI：智能解析与分类

### 商业模式
> **可持续的 SaaS 订阅模式**

| 套餐 | 价格 | 核心功能 | 目标用户 |
|------|------|---------|---------|
| Free | $0 | 基础检索 (10 次/天) | 个人用户/试用 |
| Pro | $29/月 | 无限检索 + 文件生成 | 中小企业 |
| Enterprise | $299/月 | API + 多账号 + 定制 | 大型企业 |
