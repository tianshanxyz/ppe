## 💡 建议与决策

### 技术决策

1. **前端框架**: 确认使用 Next.js 16 + React 19
   - 理由：团队熟悉，生态完善，部署简单

2. **数据库**: 确认使用 Supabase (PostgreSQL)
   - 理由：免费额度充足，自带认证，开发效率高

3. **部署方案**: Vercel (前端) + AWS (后端)
   - 理由：前端 Vercel 零配置，后端 AWS 灵活可控

### 产品决策

1. **MVP 范围**: 聚焦核心功能 (检索 + 详情 + 订阅)
   - 理由：快速验证商业模式

2. **定价策略**: Free + Pro ($29/月) + Enterprise ($299/月)
   - 理由：覆盖不同用户群体，价格有竞争力

3. **数据策略**: 优先覆盖 3 个国家 (中/美/欧)
   - 理由：主要市场，数据可获得性高
