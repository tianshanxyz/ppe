## 📖 项目简介

全球 PPE 数据获取与应用综合实施方案，旨在建设覆盖全球主要经济体的个人防护装备（PPE）数据平台，实现数据采集、治理、分析、应用的全流程管理。

### 核心目标
- 🌍 覆盖 6+ 国家（中美欧日澳加）的 PPE 数据采集
- 🧹 完整的数据治理与质量管理体系
- 🔍 5 大核心应用系统（检索、预警、文件生成、竞争力评估、数据看板）
- 🚀 生产级部署与运维保障

### 项目周期
- **总周期**: 24 周（2026-04-18 ~ 2026-10-03）
- **阶段数**: 7 个阶段
- **里程碑**: 46 个
- **总工时**: 约 1600 小时

## 🛠️ 技术栈

### 前端
- Vue 3 + TypeScript + Vite
- Element Plus (UI 组件)
- Pinia (状态管理)
- ECharts (可视化)

### 后端
- Node.js + NestJS
- PostgreSQL + TypeORM
- Redis (缓存)
- Elasticsearch (搜索)
- RabbitMQ (消息队列)

### 数据
- Puppeteer (网页抓取)
- Cheerio (数据解析)
- Python (AI 模型)

### 运维
- Docker + Docker Compose
- GitHub Actions (CI/CD)
- Prometheus + Grafana (监控)
