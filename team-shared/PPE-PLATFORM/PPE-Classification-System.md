## 1. 分类体系概述

### 1.1 分类目标

建立统一、标准、可扩展的 PPE 产品分类体系，实现：
- ✅ 全球主要市场 PPE 产品的统一分类
- ✅ 与各国法规标准的准确映射
- ✅ 支持智能检索和推荐
- ✅ 便于数据统计和分析

### 1.2 分类原则

| 原则 | 说明 |
|------|------|
| **科学性** | 基于产品本质特征，分类逻辑清晰 |
| **系统性** | 层次分明，结构完整，覆盖全面 |
| **实用性** | 便于用户理解和使用，支持业务场景 |
| **扩展性** | 预留扩展空间，适应新产品和新标准 |
| **兼容性** | 与主要国家标准分类保持兼容 |

### 1.3 分类层级

采用三级分类体系：

```
一级分类 (防护部位)
    └── 二级分类 (产品类别)
        └── 三级分类 (具体产品)
```

- **一级分类**: 6 大类，按防护的人体部位划分
- **二级分类**: 20-30 类，按产品功能和用途划分
- **三级分类**: 100+ 类，具体产品型号和规格

## 6. 分类编码规则

### 6.1 编码结构

采用层次码结构，共 4 级 10 位：

```
XX - XX - XX - XXX
│    │    │    └─ 三级分类顺序码 (001-999)
│    │    └────── 二级分类顺序码 (01-99)
│    └─────────── 一级分类顺序码 (01-99)
└──────────────── 大类前缀 (PPE)
```

### 6.2 编码示例

| 编码 | 说明 |
|------|------|
| PPE-02-01-001 | N95 口罩 |
| PPE-06-01-015 | 丁腈一次性手套 |
| PPE-01-01-003 | 矿用安全帽 |

### 6.3 扩展规则

- 预留 20% 编码空间用于新产品
- 废弃产品编码不再重用，保持历史数据稳定
- 新增三级分类需经过标准化委员会审核

## 7. 分类应用示例

### 7.1 产品归类流程

```
产品输入
    ↓
识别防护部位 → 确定一级分类
    ↓
识别产品功能 → 确定二级分类
    ↓
识别具体型号 → 确定三级分类
    ↓
匹配适用标准 → 完成归类
```

### 7.2 应用案例

#### 案例 1: 3M 8576 N95 防护口罩

**产品信息**:
- 品名：3M 8576 N95 防护口罩
- 类型：杯罩式，带呼吸阀
- 认证：NIOSH N95

**归类结果**:
- 一级分类：PPE-02 (呼吸防护)
- 二级分类：PPE-02-01 (过滤式呼吸器)
- 三级分类：PPE-02-01-01 (N95 口罩)
- 适用标准：NIOSH 42 CFR 84

#### 案例 2: Honeywell 防化手套

**产品信息**:
- 品名：Honeywell North 防化手套
- 材质：丁基橡胶
- 防护：防有机溶剂

**归类结果**:
- 一级分类：PPE-06 (手足防护)
- 二级分类：PPE-06-01 (防护手套)
- 三级分类：PPE-06-01-02 (防化手套)
- 适用标准：EN 374

### 7.3 分类检索示例

#### 场景 1: 用户需要采购 N95 口罩

**检索路径**:
1. 选择一级分类：呼吸防护
2. 选择二级分类：过滤式呼吸器
3. 选择三级分类：N95 口罩
4. 筛选条件：NIOSH 认证、带呼吸阀

**返回结果**: 所有符合 N95 标准的产品列表

#### 场景 2: 用户需要了解化学防护服标准

**检索路径**:
1. 选择一级分类：躯干和四肢防护
2. 选择二级分类：防护服
3. 选择三级分类：化学防护服
4. 查看适用标准：EN 943、GB 24540、NFPA 1992

**返回结果**: 化学防护服相关标准及法规要求
